# Project Setup

```sh
docker compose up -d
```

```sh
docker exec -it zona-app composer install
```

```sh
docker exec -it zona-app php artisan key:generate
```

```sh
docker exec -it zona-app php artisan migrate
```

```sh
npm run install
```

```sh
npm run dev
```

App Available at http://127.0.0.1:8080/
